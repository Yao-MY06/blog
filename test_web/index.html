系统概览

这是一套为大二技术方向学生设计的个人操作系统。它不是一堆待办清单，而是一套让输入→处理→输出→反馈自动运转的生产力引擎。

📥

这是什么

基于 PARA 方法论 + 项目驱动学习 + 时间块管理 + 变现验证 的个人操作系统。

"会做项目≠能赚钱。每个项目在立项前必须回答：谁需要这个？他们现在怎么解决？我能收多少钱？"
🎯

为谁设计

  • 大二计算机/技术方向学生，有觉醒焦虑
  • 想学全栈、AI、建模、动画……目标过载
  • 核心症状：拖延、娱乐依赖、无法专注、没有正反馈
💊

解决什么问题

  • 收藏夹吃灰 → 项目驱动，每个学习必须有交付物
  • B站一刷三小时 → 物理约束：屏蔽+隔离+环境控制
  • AI代劳作业 → 先手写30分钟，再看AI输出
  • 学了就忘 → GitHub绿墙+每周博客+番茄钟计数
  • 做了一堆项目，一分钱没赚到 → 变现三问强制验证

为什么需要变现层？

没有变现层的系统会让你的技术学习变成自嗨循环：

学React → 做没人用的待办应用 → 觉得不够酷 → 学Vue → 做没人用的天气应用
     ↑                                                    |
     └──────────────── 零收入，零验证，继续学下一个框架 ←──┘
立项前

谁需要这个？

确认需求存在，不是想象中的需求
开发中

他们现在怎么解决？

了解竞品和替代方案，找到差异化
交付后

明确变现路径：接单/求职/产品

回答不了这三个问题的项目 = 纯学习项目，限制投入时间（不超过总学习时间的30%）。

PARA 文件夹结构

五文件夹结构与五层OS的对应关系。

📥

00-收件箱

输入层

临时捕获：想法、链接、截图、课程通知

核心规则：24小时内清空，不存放超过一天
📓

01-日记与周期

处理层 + 反馈层

日回顾、周回顾、月回顾、博客草稿

核心规则：晚上固定时间写，不超过15分钟
🚀

02-项目管理

处理层

当前进行的项目规划、任务拆解、进度

核心规则：同时进行的项目 ≤3个
🗺️

03-领域与长期

处理层 + 变现层

技能路线图、技能决策矩阵、长期目标

核心规则：新技能必须通过决策矩阵评分（>15分才能立项）
📝

04-知识笔记

输出层

经过加工的永久笔记、代码片段、概念卡片

核心规则：用自己的话重写，不是复制粘贴
🧰

05-资源库

输入层

工具清单、模板、AI使用公约、配置备份

核心规则：配置变更后及时更新
💰

06-机会捕获

变现层

赚钱线索：需求帖、外包单、产品灵感、行业趋势

核心规则：每条线索72小时内做变现三问评估
📊

07-财务追踪

变现层 + 反馈层

收入/支出记录、报价模板、合同范本、客户反馈

核心规则：每笔交易后即时记录，每周对账，每月复盘
🗄️

Z-归档

归档

完成的项目、过期的笔记、旧回顾

核心规则：归档时写 3 句话总结

使用原则（铁律）

违反任何一条，系统效率至少下降50%。

01

项目是学习的最小单位

没有"学React"这种任务，只有"用React做一个待办应用，周三前能跑通"。

每个项目必须可演示、可放入简历。

02

同时进行的项目不超过3个

你现在觉得能学十个，实际上一个都做不完。

激进的优先级是第一生存法则。当前推荐：1个全栈项目 + 1个课程作业 + 1个恢复性活动。

03

先手写30分钟，再打开AI

任何代码、任何作业、任何笔记——先用自己的脑子和手处理30分钟，然后再用AI辅助。

AI是加速器，不是替身。

04

晚上回顾比早上计划更重要

每天花15分钟写回顾：今天做了什么、GitHub提交了吗、明天的唯一重点是什么。

没有回顾的日子等于没有活过。

05

物理约束 > 意志力

别相信自己"这次会自律"。屏蔽B站、手机放远、去图书馆、用白噪音——

让堕落变得困难，让执行变得容易。

06

回答不了变现三问的项目 = 纯学习项目

Q1 谁需要这个？ 目标用户是谁，不是"我觉得有人需要"
Q2 他们现在怎么解决？ 竞品/替代品是什么
Q3 我能收多少钱？ 接单定价/求职薪资/产品收入

回答不了 → 允许做，但限制投入时间≤总学习时间的30%，且不占用早晨7:00-8:30的黄金时间块。

07

技术学习必须指向三种变现路径之一

💼 接单

能独立完成小型项目后

猪八戒/码市/朋友推荐，从500元小单开始
🎯 求职

有2-3个作品集项目后

投递实习/校招，薪资是能力的市场定价
🚀 产品

发现真实需求且有能力解决

小工具/SaaS/付费模板，独立收费

大三上学期结束时，你必须已经赚到第一笔技术收入（哪怕是100元的外快）。

"你不是没时间，你是没优先级。你不是懒，你是没系统。你不是在学技术，你是在学赚钱的手段——别假装自己在搞艺术。"
